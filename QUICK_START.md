## 🚀 Deploy în 3 Pași

### 1. Install Dependencies
```bash
cd apps/web
pnpm install
```

### 2. Build & Test
```bash
pnpm build
pnpm start
# Open http://localhost:3000
```

### 3. Deploy
Upload to Vercel/Netlify/Railway cu HTTPS activat (necesar pentru PWA)
