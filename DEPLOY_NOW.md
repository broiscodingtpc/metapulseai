## ⚡ Deploy în 5 Minute

### Pas 1: Install & Build
```bash
cd apps/web
pnpm install
pnpm build
```

### Pas 2: Test Local
```bash
pnpm start
# Open http://localhost:3000
```

### Pas 3: Lighthouse Check
```bash
npx lighthouse http://localhost:3000 --view
# Verifică că toate scorurile sunt 90+
```

### Pas 4: Deploy Production
```bash
# Vercel
vercel --prod

# Sau Netlify
netlify deploy --prod

# Sau Railway
railway up
```

### Pas 5: Test PWA

- Install app pe mobile
- Test offline mode
